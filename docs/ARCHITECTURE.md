## ğŸ¯ Design Decisions

### Why Separate World Files?

**Problem:** Having both Position and Velocity controllers in same robot causes overriding.

Position controller uses PID â†’ continuously applies effort â†’ overrides velocity commands.

**Solution:** Create separate SDF files:
- `heal_robot_position.sdf` - Only JointPositionController (x6)
- `heal_robot_velocity.sdf` - Only JointController (x6)

### Why Custom Bridge Instead of ros_gz_bridge?

**Problem:** Standard ros_gz_bridge requires complex YAML configuration for 12 topics.

**Solution:** Simple Python bridge:
- Single ROS2 array topic â†’ 6 Gazebo topics
- Easy to understand and modify
- Matches real robot interface

### Why Fixed Gripper Joints?

**Problem:** DART physics doesn't support mimic constraints (Robotiq gripper uses mimic joints).

**Solution:** Convert gripper joints to fixed type - gripper stays in default pose but no physics errors.
